Installationsinstruktioner
 Microsoft JDBC Driver 6.0 för SQL Server


INTRODUKTION
------------
Det här är instruktioner för hur du packar upp sqljdbc_<version>_enu.exe och sqljdbc_<version>_enu.tar.gz.


WINDOWS
-------
1. Läs filen license.txt. Om du laddar ned och använder Microsoft JDBC Driver 6.0 för SQL Server innebär det att du accepterar den här licensen.

2. Ladda ned sqljdbc_<version>_enu.exe och spara filen i en tillfällig katalog.

3. Kör sqljdbc_<version>_enu.exe

4. Ange en installationskatalog när du uppmanas till det. Det rekommenderas att du packar upp
 den här zip-filen i %ProgramFiles% med standardkatalogen:

	Microsoft JDBC DRIVER 6.0 för SQL Server


UNIX
----
1. Läs filen license.txt. Om du laddar ned och använder Microsoft JDBC Driver 6.0 för SQL Server innebär det att du accepterar den här licensen.

2. Ladda ned sqljdbc_<version>_enu.tar.gz och spara filen i en tillfällig katalog.

3. Om du vill packa upp den zippade tar-filen går du till den katalog där du vill packa upp
 drivrutinen och skriver:

	gzip -d sqljdbc_<version>_enu.tar.gz

4. Om du vill packa upp tar-filen flyttar du den till den katalog där du vill installera drivrutinen
 och skriver:

	tar -xf sqljdbc_<version>_enu.tar
